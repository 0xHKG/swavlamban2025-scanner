## Personalization Guide

Replace these placeholders when sending to each organization:

| Placeholder | Example | Description |
|-------------|---------|-------------|
| `[Organization Name]` | DRDO, HAL, IIT Delhi | Full organization name |
| `[USERNAME]` | drdo, hal, iitdelhi | Login username (lowercase, no spaces) |
| `[PASSWORD]` | Secure123! | Initial password (min 8 chars) |
| `[NUMBER]` | 50, 100, 200 | Maximum entries allowed |
| `[Pass types list]` | Exhibition Day 1 & 2, Interactive Sessions | Comma-separated list of allocated pass types |

---

## Pass Types Reference

When filling `[Pass types list]`, use combinations of:
- **Exhibition Day 1** (25 Nov, 1100-1730 hrs)
- **Exhibition Day 2** (26 Nov, 1000-1730 hrs)
- **Interactive Sessions** (26 Nov, Session I: 1030-1130, Session II: 1200-1330)
- **Plenary Session** (26 Nov, 1530-1615 hrs)

**Example allocations:**
- Standard Exhibitor: "Exhibition Day 1 & 2"
- Panel Speaker: "Exhibition Day 2, Interactive Sessions"
- VIP Attendee: "All passes (Exhibition Day 1 & 2, Interactive Sessions, Plenary)"
- Session-specific: "Interactive Sessions, Plenary Session"

## Tips for Sending

1. **WhatsApp Formatting**:
   - Use `*bold*` for headers (asterisks)
   - Use emojis for visual appeal
   - Keep lines short for mobile readability

2. **Timing**:
   - Send during business hours (0900-1700)
   - Allow organizations 2-3 days to register attendees
   - Send reminder 1 week before event

3. **Follow-up**:
   - Monitor login activity in Admin Panel
   - Follow up with organizations that haven't logged in after 2 days
   - Offer assistance via phone/email if needed

4. **Security**:
   - Send credentials via secure channel (official WhatsApp/Email)
   - Remind users to change password immediately
   - Never share credentials in public groups
